# AI Video Assistant 

An AI-powered Video Assistant that processes videos, generates transcripts, and enables conversational Q&A using RAG and Mistral AI.

## Features

* YouTube audio extraction using yt-dlp
* Audio preprocessing and chunking
* Whisper transcription
* Translation support
* RAG pipeline (Coming Soon)
* Mistral AI integration (Coming Soon)

## Tech Stack

* Python
* Whisper
* Mistral AI
* yt-dlp
* pydub
